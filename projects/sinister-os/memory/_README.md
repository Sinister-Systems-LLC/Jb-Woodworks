# memory/ — Sinister OS lane memory

> **Author:** RKOJ-ELENO :: 2026-05-24

Every per-project EVE session opened on `projects/sinister-os/` appends durable session learnings here. Cross-session knowledge that other lanes need still goes to `_shared-memory/knowledge/` (the global brain) — this folder holds notes specific to *this* project's work.

## Files

- `decisions.md` — every architectural decision with a date + rationale. Append-only.
- `gotchas.md` — every "I tried X, it broke because Y, fix is Z" learning. Append-only.
- `vm-snapshots.md` — VM names + snapshot names + what state they represent.
- `package-pins.md` — pinned package versions that we know work; reason for the pin.
- `proton-titles.md` — operator's owned game titles + their proton status as we verify them.
