## How to generate

### Banner (one-click)

```cmd
generate-banner.bat
```

Drops `generated/banner-v1.png` + `generated/banner-v1.png.meta.json`.

### One-off ad-hoc

```cmd
python -m nano_banana ^
  --prompt "Half-body portrait of the JKOR demon-jester holding cards" ^
  --output "generated/portrait-cards.png" ^
  --brand jkor ^
  --ref "reference/00-base-banner-original.png"
```

The `--brand jkor` flag auto-appends the locked style suffix. `--ref` style-transfers from the reference image so the character stays consistent.

### From Python

```python
from nano_banana import jokr_image
result = jokr_image(
    prompt="The demon-jester throwing a card mid-air, motion blur on the card",
    output_path="generated/card-throw.png",
    ref_images=["reference/00-base-banner-original.png"],
)
print(result.status, result.output_path)
```
