Created the first proper SMPL brand pack:

- `logos/` — 12 wordmark / lockup variants (primary horizontal, spotlight cone, truss, stacked, cinematic banner, watermark, mono white / black / gold, outline, vertical, flat)

- `print/` — 2 business-card templates (front, back)
- Docs: README, TOKENS, USAGE, CHANGELOG

### Exploratory directions (proposals — operator review before public use)

Three new visual languages introduced, each ADDS to the existing identity rather than replacing it:

1. **Spotlight cone** (`logos/02-spotlight-cone.svg`) — the canonical triangle accent extended into a volumetric stage-light cone. Optional hero treatment.
2. **Truss** (`logos/03-truss.svg`) — wordmark on a stylized box-truss element. Industry surfaces only.
3. **Stamp / Seal family** (`marks/01-emblem-seal.svg`, `03-mark-hex.svg`, `04-mark-shield.svg`, `05-mark-tour-case.svg`) — secondary marks with non-rectangular containers (circle / hex / shield / case).

**Doctrine (operator 2026-05-23):** "All logo and branding is based on the image these companies already have and does not change them fully — but enhances them in ways." Honored. None of the canonical files in `../public/img/` were altered. Exploratory variants live here for review.
