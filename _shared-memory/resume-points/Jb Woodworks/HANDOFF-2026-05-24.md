## Architecture (current production state)

```
User browser
   ↓ DNS at Vercel (registrar): @ ALIAS + www CNAME → cname.vercel-dns.com
Vercel edge (SSL terminates with CN=jbwoodworks.co, LE R12, valid until 2026-07-20)
   ↓ vercel.json: redirect www.* → apex (308), rewrite /* to Railway
Railway service `web` (id 79cb641a-8ce3-4f91-b9bc-3fbfe20f96ed)
   ↓ Next.js 15.5 prod bundle
Railway Postgres (id 4951c796-d95c-488f-af94-024c2c47300a) ← ContactInquiry persists here
```

**Why Vercel-proxy-in-front-of-Railway:** Railway's cert provisioning for `jbwoodworks.co` got stuck `VALIDATING_OWNERSHIP` for 90+ min and never recovered (suspected Let's Encrypt rate-limit hit during earlier debugging). Vercel issued a fresh LE cert immediately. Railway runs the actual app; Vercel is just SSL+CDN. The decision is documented in `projects/jb-woodworks-proxy/README.md` in the Sanctum monorepo.
